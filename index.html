Project

📦 Container Tracking System (IoT Project)

This project is a real-time IoT-based container tracking system designed to monitor environmental and security conditions inside containers.

It uses multiple components such as ESP32 microcontroller, GPS module for live location tracking, gas sensor for leakage detection, temperature sensor, humidity sensor, and LDR sensor for light detection.

All sensor data is collected and processed through ESP32 and sent via API integration. The system is connected with a Telegram Bot to provide real-time alerts instantly when abnormal conditions are detected.

This project helps in improving logistics safety, monitoring cargo conditions, and ensuring real-time tracking of containers efficiently.
